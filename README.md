## Dinodex

Welcome to the Dinodex!

Dinodex is an interactive terminal based dinosaur pokedex built on vikiru's [RESTasaurs](https://github.com/vikiru/restasaurus) API & [Typer](https://github.com/fastapi/typer)

Collect dino ascii's, and share them with your friends. 

View a gallery of your dinos, and learn more about these beasts of yore!

## Compile & Build
**REQUIRES** 
[uv](https://github.com/astral-sh/uv)

run the included quickbuild.sh file to build you dinodex
